## 仕様
以下に雑に描いたシステム図とピンの割り当てを示す。
TIMのCHxやADCのINx,buttonxは番号の通りそれぞれモータ1,2,3,4に対応するものとする。
TIM_CHx⇒モタドラ用PWM出力
ADC_INx⇒モーター出力調整用可変抵抗用
buttonx⇒モーター動作(押してる間に動作)
